📊 Istruzioni per Importare le Macro VBA

Guida step-by-step per aggiungere le macro di formattazione al tuo file Excel

📥 Download File Macro

Prima di iniziare, scarica il file delle macro:

📁 Scarica FormattazioneProspettoFerie.bas
⚠️ Importante: Devi prima scaricare questo file BAS, poi seguire la sequenza di importazione sul file Excel (.xlsm) che hai già scaricato.
1 Apri il file Excel scaricato

Individua il file Prospetto_Ferie_[Anno].xlsm nella cartella dei download e aprilo con Microsoft Excel.

Nota: Se Excel mostra un avviso di sicurezza sulle macro, clicca su "Abilita contenuto" per permettere l'uso delle macro.
2 Apri l'Editor VBA

Premi la combinazione di tasti:

Alt + F11

Oppure vai nel menu Sviluppo > Visual Basic

Se non vedi il menu "Sviluppo":
1. Vai su File > Opzioni
2. Seleziona "Personalizzazione barra multifunzione"
3. Spunta la casella "Sviluppo" nella colonna di destra
3 Importa il modulo VBA

Nell'Editor VBA:

  1. Fai clic destro sul nome del progetto (nella finestra "Progetto")
  2. Seleziona Importa file...
  3. Naviga fino alla cartella dove hai scaricato il file FormattazioneProspettoFerie.bas
  4. Seleziona il file e clicca Apri
4 Verifica l'importazione

Dopo l'importazione, dovresti vedere nella finestra "Progetto" un nuovo modulo chiamato FormattazioneProspettoFerie.

5 Salva il file

Premi Ctrl + S o vai su File > Salva per salvare il file con le macro importate.

✅ Perfetto! Le macro sono state importate correttamente.
6 Chiudi l'Editor VBA

Chiudi l'Editor VBA premendo Alt + Q o cliccando sulla X per tornare al foglio Excel.

🎯 Funzioni Macro Disponibili

Una volta importate le macro, avrai accesso alle seguenti funzioni di formattazione:

  • FormattaTuttiIFogli: Applica tutte le formattazioni a tutti i fogli
  • ApplicaSoloColoriRossi: Colora solo weekend e festività in rosso
  • ApplicaSoloColoriVerdi: Colora solo le tipologie di ferie (F, FV, FS, R, MF) in verde
  • ApplicaSoloBordi: Applica solo i bordi neri
  • RimuoviFormattazione: Rimuove tutta la formattazione

Per eseguire una macro: Premi Alt + F8, seleziona la macro desiderata e clicca "Esegui".

7 Esegui la macro FormattaTuttiIFogli

Una volta caricato il file, la macro da lanciare è quella denominata "FormattaTuttiIFogli".

Per applicare immediatamente tutte le formattazioni:

  1. Premi Alt + F8 per aprire la finestra delle macro
  2. Seleziona FormattaTuttiIFogli dall'elenco delle macro disponibili
  3. Clicca Esegui
  4. Conferma l'operazione quando richiesto
🎉 Eccellente! Il tuo prospetto ferie è ora completamente formattato!
💡 Suggerimento: La macro FormattaTuttiIFogli è quella principale che applica tutte le formattazioni necessarie al prospetto ferie. È la macro raccomandata per l'uso normale.

🔧 Risoluzione Problemi

  • "Impossibile importare il file": Verifica che il file FormattazioneProspettoFerie.bas sia stato scaricato correttamente
  • "Macro disabilitate": Vai su File > Opzioni > Centro protezione > Impostazioni Centro protezione > Impostazioni macro e seleziona "Abilita tutte le macro"
  • "Errore durante l'esecuzione": Assicurati che il file Excel sia in formato .xlsm (macro-enabled)
