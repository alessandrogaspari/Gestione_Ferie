🌐SOLUZIONE NGROK

Problema Risolto

  • Limite: Ngrok cambia URL ad ogni riavvio
  • Soluzione: Automazione con GitHub Pages come punto di accesso fisso

Configurazione Ngrok

1. Installazione

# Download da https://ngrok.com/download
# Estrazione in C:\ngrok

2. Registrazione Token

ngrok config add-authtoken IL_TUO_TOKEN_NGROK

3. Avvio Tunnel

ngrok http 3000

4. Configurazione Avanzata

  • File: ngrok.yml
  • Posizione: %USERPROFILE%\.ngrok2\ngrok.yml
  • Contenuto:
version: "2"
authtoken: IL_TUO_TOKEN
tunnels:
  appferie:
    proto: http
    addr: 3000
    bind_tls: true
